Help me ship a release on Heroboard.

1. Identify the project (from `$ARGUMENTS` or `list_projects`).
2. Summarize what's ready to ship (tasks in the done columns for the open release) using `list_tasks`.
3. Note that shipping is an admin-only action done in the Heroboard web app (Releases → Ship) — don't attempt to mutate it over MCP. Give me the direct link to the project's Releases page.
